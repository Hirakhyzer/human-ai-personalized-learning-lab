# Ethics, privacy, and deployment boundary

## Core boundary

This repository does not contain real student data and does not make real educational decisions. It is a research scaffold for studying learning analytics and Human-AI collaboration.

## Prohibited use without formal approval

- Grading real students.
- Tracking identifiable student behavior.
- Assigning interventions without educator review.
- Making high-stakes placement or progression decisions.
- Claiming fairness for a real student population from synthetic results.

## Requirements for real studies

A real deployment would require informed consent or a lawful/approved basis, minimization of personal data, privacy review, accessibility review, educator oversight, opt-out procedures, data retention controls, and ethics/IRB-style review where applicable.

## Human review

The software can generate recommendations for study design and synthetic analysis. For real learners, teachers/researchers must review and adapt any intervention.
